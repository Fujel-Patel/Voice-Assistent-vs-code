# JARVIS — Agent Skills Index

Skills are specialized procedures that AI agents should follow when performing common operations on this codebase.

## Available Skills

| Skill | File | When to Use |
|-------|------|-------------|
| Refactor | [refactor.md](refactor.md) | Restructuring modules, extracting classes, moving files |
| Debug | [debug.md](debug.md) | Diagnosing runtime errors, pipeline failures, audio issues |
| Code Review | [code-review.md](code-review.md) | Reviewing PRs, auditing code quality, enforcing standards |

## Using Skills

1. Read the relevant skill file before starting work
2. Follow the checklist in order — don't skip steps
3. Run verification commands listed in each skill
4. Update `agent/tasks.md` when work is complete
